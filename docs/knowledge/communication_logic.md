## Serial Communication & Hardware Settings (Mega 2560 + ESP8266)

### 1. RX / TX の役割
Mega 2560とESP8266は「シリアル通信」でデータをやり取りしている。
- **TX (Transmit):** 送信ピン。データを出力する。
- **RX (Receive):** 受信ピン。データを受け取る。
- **基本ルール:** 相手のTXを自分のRXに接続する（クロス接続）。

### 2. ジャンパーキャップの意味
ESP8266を操作するための物理的な「ON/OFFスイッチ」。
- **接続時:** Mega 2560とESP8266をシリアル通信ラインで連結し、Wi-Fi通信を可能にする。
- **切断時:** 物理的に分離する。プログラム書き込み時にシリアル信号が干渉してエラーになるのを防ぐため、書き込み時は外す必要がある。

### 3. SoftwareSerial (A8 / A9) の選択理由
Mega 2560のハードウェアシリアルポートはデバッグ用に温存されているため、空きピンであるA8/A9を使い、ソフトウェア的に擬似シリアルポートを構築して通信を行っている。
